# List Directory Files

Lists all files and directories in a specified path or current directory.

## Usage



If no path is provided, lists current directory.

## Examples



## Output Format

Shows:
- File/directory names
- Type (file/dir)
- Size (for files)
- Last modified date

## When to Use

- User asks to list files
- Need to see directory contents
- Exploring file structure
- Verifying file existence
