Master AI agent development with both major frameworks — side by side.

AI agents are no longer chatbots. They perceive, reason, and act — sending emails, writing code, searching the web, and executing complex workflows autonomously. This is the most significant shift in software engineering since cloud computing.

Agentic AI is the first book to cover both the OpenAI Agents SDK and the Claude Agent SDK in depth, teaching you to build production-ready autonomous systems from scratch.

What You'll Learn

  • Build AI agents that use tools, make decisions, and complete tasks autonomously
  • Design multi-agent systems with 6 orchestration patterns (triage, manager, sequential, parallel, evaluator, router)
  • Implement guardrails, security layers, and human-in-the-loop controls
  • Create voice agents with VoicePipeline and the Realtime API
  • Use the Claude SDK's unique Skills system and parallel subagents
  • Connect agents to external services via MCP (Model Context Protocol)
  • Deploy to production with FastAPI, error handling, logging, and monitoring
  • Defend against prompt injection, tool abuse, and data leakage

Four Complete Projects

Each implemented in both frameworks so you can compare approaches:
  1. Research Assistant — multi-agent web research with structured reports
  2. Customer Service System — triage routing, memory, streaming, approvals
  3. Autonomous Daemon — background task scheduler with React UI
  4. Voice Agent — real-time voice with agent handoffs

Software engineers, backend developers, technical leads, and ML engineers who want to build AI agent systems — not just use them. Requires basic Python knowledge. No ML or deep learning experience needed.
